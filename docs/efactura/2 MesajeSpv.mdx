# Vizualizarea mesajelor SPV

Pentru vizualizarea mesajelor SPV,  se deschide ecranul dedicat din pagina "Urmarire" pe butonul corespunzator, ca in imaginea de mai jos.

![Vizualizare mesaje SPV](../efactura/Imagini%20eFactura/mesajespv.png)

Operatiunile care pot fi efectuate pe aceasta pagina sunt in general usor de intuit se le vom clarifica in detaliu in sectiuni dedicate daca veti consiera necesar.

## Principala functionalitate a paginii este vizulizarea facturii in format PDF.

Dupa cum se vede clar si in imaginea de mai sus, se poate selecta factura dorita din lista de facturi disponibile si se poate apasa butonul "PDF" pentru a deschide factura in format PDF.

![Vizualizare mesaje SPV PDF](../efactura/Imagini%20eFactura/mesajespvpdf.png)
