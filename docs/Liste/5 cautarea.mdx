# Căutarea in conținut

Cautarea se realizeaza in toate coloanele posibile, chiar si cele care
nu sunt vizibile la momentul cautarii. Rezultatul este setul de date
FILTRAT dupa textul introdus in caseta de cautare.

Textul cautat este marcat cu galben.

Daca rezultatul cautarii este nul, caseta de cautare devine rosie
