 # Selectarea coloanelor vizibile
 
 Implicit, listele returnează un set de coloane destul de extins, astfel încât să se poată genera rapoarte relevante diferite, pe baza aceluiași set de date.

 Pentru a modifica setul initial de coloane vizibile, se face click pe butonul corespunzator din bara de instrumente (vezi mai jos).

 Apoi se poate trage un cap de coloană în sau din panoul de coloande disponibile.

<div style={{ textAlign: 'center', padding: '30px' }}>
   <ColumnChooserImage imgMaxWidth={'100%'} />
</div>
Avem practic un panou unde punem coloanele ca sa le ascundem si de unde le putem scoate din nou daca avem nevoie.

Coloanele rămase vizibile pot fi redimenionate si ordonate.
