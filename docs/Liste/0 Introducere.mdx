# Ghid pentru modulul Liste

## Introducere

Modulul aduce in interfata utilizatorului maxim de informatie, in functie de drepturile de acces si de selectia facuta.

Apoi, utilizatorul are la dispozitie mai multe posibilitati de filtrare si sortare a datelor, pentru a obtine informatiile dorite.

Se pot salva criteriile de filtrare si sortare, pentru a putea fi folosite ulterior.

Detaliile de functionare sunt prezentate si in helpul disponibil in interfata, fiind accesibil din pagina Rapoarte / Liste, prin butonul “?” din dreapta sus.
