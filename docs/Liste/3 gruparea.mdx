# Gruparea pe coloană

Pentru a grupa după o coloană, **_se trage capul de coloană_** deasupra tabelului, in zona de grupare.
<em>
  Unde scrie: Trageți un antet de coloană aici pentru a grupa după acea coloană
</em>
<br/>
<br/>


<div>

Acolo unde este posibil, in general la coloanele de tip numeric sau la
cele ce exprimă timpi se vor calcula automat sume sau alte agregate în
funcție de coloanele grupate.

Gruparea are sens când sunt mai multe înregistrări cu

<strong>o valoare comună</strong>. De exemplu pentru un raport de timesheet,
gruparea după un client sau un contract, samd.

Se pot face grupări după mai multe coloane.
