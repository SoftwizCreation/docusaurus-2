# Șabloane de formate
Probabil ca vor fi criterii preferate de selectie, ordonare si grupare la anumite liste. 

Pentru a evita repetarea acelorasi operatiuni de formatare se pot salva formatele pentru utilizare ulterioară.

Practic, programul vine cu un set brut de coloane READ-ONLY, care pot fi formatate după preferințele utilizatorului si salvate separat 
pentru a fi folosite ulterior.
